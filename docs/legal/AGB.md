## 2. Vertragsgegenstand

### 2.1 Leistungsbeschreibung

Die App bietet folgende Funktionen:

1. **Video-Analyse mit MediaPipe**
   - Echtzeit-Analyse von Körperhaltung, Blickkontakt, Mimik und Gestik
   - Lokale Verarbeitung im Browser (keine Uploads von Videos)
   - Aggregierte Metriken-Übertragung an Server

2. **KI-basiertes Feedback**
   - Automatische Interpretation der Analysedaten
   - Personalisierte Verbesserungsvorschläge
   - Powered by OpenAI GPT

3. **Account-Verwaltung**
   - Registrierung mit E-Mail und Passwort
   - Sichere Authentifizierung (JWT)
   - Analyse-Historie

4. **Subscription-Modell**
   - Free Tier: Basis-Features
   - Pro Tier: Erweiterte Features
   - Zahlungsabwicklung via Stripe

### 2.2 Technische Voraussetzungen

**Mindestanforderungen:**
- Moderner Webbrowser (Chrome 90+, Firefox 88+, Safari 14+)
- Webcam mit mindestens 720p Auflösung
- Internetverbindung mit mindestens 1 Mbit/s
- JavaScript aktiviert

**Empfohlen:**
- 1080p Webcam
- 10 Mbit/s Internetverbindung
- Desktop-Gerät (Laptop/PC)
